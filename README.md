# G+1 Residential Building – Revit BIM Project

## Project Overview
This project demonstrates BIM modeling and quantity estimation of a G+1 residential building using Autodesk Revit.

## Project Features
• Ground Floor Plan  
• First Floor Plan  
• Building Elevations  
• Building Sections  
• 3D Visualization  
• Door Schedule  
• Window Schedule  
• Room Schedule  
• Wall Quantity Estimation  

## Software Used
Autodesk Revit

## Skills Demonstrated
• BIM Modeling  
• Architectural Planning  
• Quantity Takeoff  
• Construction Documentation
